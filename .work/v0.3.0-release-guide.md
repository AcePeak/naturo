## 1. 新功能 (Features)

### 1.1 Unified Selector 引擎 (#100)
用类似 CSS 的语法精确定位 UI 元素。

```bash
# 按类型+名称
naturo click "Button:Save"

# 按 AutomationId
naturo click "#saveButton"

# 组合条件
naturo click "Button:Save[enabled=true]"

# 层级选择
naturo click "Window:Notepad > Edit"
```

### 1.2 命令合并：window → app (#170)
所有窗口操作合并到 `app` 命名空间。

```bash
# 旧 (v0.2.1): naturo window focus "Notepad"
# 新 (v0.3.0):
naturo app focus "Notepad"
naturo app close "Notepad"
naturo app minimize "Notepad"
naturo app maximize "Notepad"
naturo app restore "Notepad"
naturo app move "Notepad" --x 100 --y 200
naturo app move "Notepad" --width 800 --height 600
naturo app windows                    # 列出所有窗口
naturo app windows --app "Notepad"    # 指定 app 的窗口

# 向后兼容：window 命令仍可用，显示 deprecation 提示
naturo window focus "Notepad"
# Expected: 功能正常 + deprecation warning
```

### 1.3 命令合并：hotkey → press (#172)
所有快捷键操作合并到 `press` 命令。

```bash
# 旧 (v0.2.1): naturo hotkey "ctrl+s"
# 新 (v0.3.0):
naturo press "ctrl+s"
naturo press "alt+f4"
naturo press "enter"

# 向后兼容：hotkey 命令仍可用
naturo hotkey "ctrl+s"
# Expected: 功能正常 + deprecation warning
```

### 1.4 snapshot 命令从顶层 help 隐藏 (#163)
```bash
naturo --help
# Expected: 不显示 snapshot

# 但命令仍然可用：
naturo snapshot list --help
# Expected: 正常显示帮助
```

### 1.5 Chrome / Electron CLI 命令移除 (#262, #139)
v0.2.0 计划中的移除，v0.3.0 执行。

```bash
naturo chrome
# Expected: "No such command 'chrome'"

naturo electron
# Expected: "No such command 'electron'"
```

### 1.6 Post-Action 验证引擎 (#231)
click/type/press 后自动验证操作是否成功。

```bash
# click 后返回验证结果
naturo --json click "Button:OK"
# Expected JSON 包含 "verified": true/false/null

naturo --json type "Hello" --app Notepad
# Expected JSON 包含 "verified": true/false/null
```

### 1.7 --see 旗标：操作后自动截图 (#174)
```bash
naturo click "Button:Save" --see
# Expected: 执行 click 后自动截取 UI 快照，输出截图路径

naturo type "Hello" --see
# Expected: 输入后自动截取 UI 快照
```

### 1.8 Capture 支持元素引用和区域裁剪 (#160)
```bash
# 先获取元素引用
naturo see --app Notepad

# 用 ref 截图
naturo capture --ref e5 --app Notepad
# Expected: 只截取 e5 元素区域

# 区域裁剪
naturo capture --region 100,200,400,300
# Expected: 截取指定坐标区域
```

### 1.9 Cascading Recognition: UIA → CDP → AI Vision (#140)
`see` 命令自动选择最佳识别方式。

```bash
# 普通 Win32 app → UIA
naturo see --app Notepad
# Expected: UIA 元素树

# Electron app → 自动尝试 CDP
naturo see --app "Visual Studio Code"
# Expected: 尝试 CDP，如不可用回退 UIA

# AI Vision 兜底
naturo see --app SomeApp --provider anthropic --model claude-sonnet-4-20250514
# Expected: 截图 + AI 识别 UI 元素
```

### 1.10 Electron App 自动检测 (#138)
```bash
naturo see --app "Visual Studio Code"
# Expected: 检测到 Electron，提示 CDP 可用
```

### 1.11 Global AI Flags (#142)
```bash
naturo see --provider anthropic --model claude-sonnet-4-20250514 --api-key sk-xxx --app Notepad
naturo find "Save button" --provider openai --model gpt-4o
# Expected: 使用指定的 AI provider 进行识别
```

### 1.12 Anthropic 订阅/OAuth Token 支持 (#143)
```bash
# 环境变量方式
ANTHROPIC_API_KEY=sk-ant-xxx naturo see --provider anthropic --app Notepad
# Expected: 使用 Anthropic API 进行 AI 识别
```

### 1.13 Snapshot Session 隔离 (#173)
多个并发 agent 使用独立的 snapshot session。

```bash
naturo see --app Notepad --session agent1
naturo see --app Calculator --session agent2
# Expected: 两个 session 互不干扰，元素引用 (eN) 各自独立
```

### 1.14 自动安装可选依赖 (#148)
```bash
# 首次运行 AI 相关命令时自动安装依赖
naturo see --provider anthropic --app Notepad
# Expected: 如果缺少 anthropic 包，自动提示/安装
```

### 1.15 Fuzzy Command Matching (#149)
```bash
naturo clik
# Expected: Did you mean 'click'?

naturo captur
# Expected: Did you mean 'capture'?
```

### 1.16 Wait 命令简化 (#169)
```bash
naturo wait 3
# Expected: 等待 3 秒（不需要 --seconds 参数）

naturo wait 0.5
# Expected: 等待 0.5 秒
```

### 1.17 CLI 参数标准化 (#166)
统一所有命令的参数命名风格。

```bash
# --app 统一用于指定目标应用
naturo click "Button:OK" --app Notepad
naturo type "Hello" --app Notepad
naturo see --app Notepad
naturo find "button" --app Notepad
# Expected: 所有命令都支持 --app
```
