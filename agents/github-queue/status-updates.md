# Issue Status Updates Queue

Dev-Sirius writes issue status changes here when GitHub tools are unavailable.
Orc-Mycelium processes them and applies the changes on GitHub.

Format:
```
## Issue #N
- **Action**: add label X / remove label Y / comment / assign
- **Comment**: <message text>
- **Date**: YYYY-MM-DD
- **Status**: pending | done
```
